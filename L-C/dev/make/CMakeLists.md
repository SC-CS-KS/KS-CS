# CMakeLists
```md
语法由命令、注释和空格组成，其中命令是不区分大小写的。
符号 # 后面的内容被认为是注释。
命令由命令名称、小括号和参数组成，参数之间使用空格进行间隔。
```

## 命令
* FIND_PACKAGE
```md
FIND_PACKAGE( <name> [version] [EXACT] [QUIET] [NO_MODULE] [ [ REQUIRED | COMPONENTS ] [ componets... ] ] )
```
