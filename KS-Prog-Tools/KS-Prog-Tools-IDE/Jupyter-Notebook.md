#  [Jupyter Notebook](https://jupyter.org/)
```md
是一个 Web 应用程序，便于创建和共享文学化程序文档，支持实时代码、数学方程、可视化和 Markdown，
其用途包括数据清理和转换、数值模拟、统计建模、机器学习等等。
目前，数据挖掘领域中最热门的比赛 Kaggle 里的资料都是 Jupyter 格式。
```
## Install
```bash
$ pip3 install jupyter

$ jupyter notebook
```
